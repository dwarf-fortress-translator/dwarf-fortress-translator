# Dwarf Fortress OCR Translator

**Dwarf Fortress OCR Translator** is a real-time text translation tool for Dwarf Fortress. It captures text from the screen using OCR (Optical Character Recognition) and displays the translation in a semi-transparent overlay above the game window.

## 🎯 Purpose

This program is specifically designed for translating Dwarf Fortress. It creates a translation window overlay on top of the game. It uses **Tesseract OCR 5** with custom fine-tuning for the game's pixel font.

## 🚀 Features

- **Instant translation** – Select an area on screen (`Ctrl+Alt`) and the translation appears directly above the game.
- **DFHack integration** – Automatically reads `.md` files generated by DFHack for perfect translation of units, items, and artifacts.
- **Two OCR engines**:
  - **Tesseract** – Local, fast, works with custom trained font.
  - **Gemini API** – ⚠️ Under development, not recommended for use.
- **20+ Languages** – Translate to popular languages (Russian, Ukrainian, German, French, Japanese, Korean, etc.).
- **Translation history** – All translations are saved to a database with search and export.
- **Flexible overlay settings** – Customize font, colors, opacity, size, and display time.

## 🖼️ System Requirements

- **Windows** 10/11 (Linux/Mac may work but not tested)

- **DFHack** (required for automatic text export from the game)
- **Python 3.8+** (source version only)

## 📦 Installation & Setup

### Step 1: Configure DFHack (in-game command)

Open the DFHack console in Dwarf Fortress and run the following command:
```bash
keybinding set Ctrl-T@dwarfmode/ViewSheets/UNIT|dwarfmode/ViewSheets/ITEM "markdown my_units -o"
```
### Step 2: Save unit/item description in-game

Select a unit, item, or artifact and press **Ctrl+T** at least once. This saves the description to `markdown_my_units.md`. The game will automatically overwrite this file each time you press the hotkey on a new unit or item.

### Step 3: Install the program

Download the program to any location, extract, and run the `Dwarf Fortress Translator.exe` file.

### Step 4: Configure the program

- Go to the **General** tab.
- Select your target language.
- Specify the path to `markdown_my_units.md`.  
  **Example path:**  
  `C:\Program Files (x86)\Steam\steamapps\common\Dwarf Fortress\markdown_my_units.md`

![Settings](https://raw.githubusercontent.com/dwarf-fortress-translator/dwarf-fortress-translator/main/setting.jpg)

### Step 5: Translate selected area (Ctrl+Alt)

Press **Ctrl+Alt** – the screen freezes, and you can select any area you want to translate.

- [Selection example](https://ibb.co/rRnF2XPy)
- [Translation result example](https://ibb.co/FLSxdwJP)

### Step 6: Quick translation of saved description

**Workflow:**
1. In the game, select a unit or item.
2. Press **Ctrl+T** – DFHack saves the description to `markdown_my_units.md`.
3. Press **Ctrl+Num0** – the program translates the file and displays the result in the overlay.

## ⌨️ Hotkeys

| Hotkey | Action |
|:---|:---|
| `Ctrl+Alt` | Select screen area for translation |
| `Ctrl+Shift+2` | Show/hide settings window |
| `Ctrl+T` (in game) | Save unit/item description to `.md` file (requires DFHack) |
| `Ctrl+Num0` | Translate `markdown_my_units.md` file into overlay |

## ⚙️ Overlay Settings (Overlay Tab)

| Setting | Description |
|:---|:---|
| **Show original text** | Show original text alongside translation |
| **Font Family** | Choose font for text display |
| **Font Size** | Font size (8–30) |
| **Background** | Overlay background color |
| **Original text** | Original text color |
| **Translated text** | Translated text color |
| **Border** | Overlay border color |
| **Opacity** | Overlay transparency (30–100%) |
| **Overlay max width** | Maximum overlay width (300–1000 pixels) |
| **Overlay max height** | Maximum overlay height (100–800 pixels) |
| **Auto height** | Automatically adjust height to fit text |
| **Display time** | Translation display duration in seconds (1–300) |
